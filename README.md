# BackupService
## Cлужба Windows для резервного копирования директории в архив
### Настройка
Рядом с исполняемым файлом положить файл `service_config.txt`, в который записать следующие параметры:\
В 1-ой строке - путь к директории\
В 2-ой строке - путь к выходному архиву\
В последующих строках - имена (маски) файлов внутри директории, подлежащих копированию\
Маска допускает следующие обозначения:\
? - ровно один символ\
\* - 1 или более символов
### Использование
```
BackupService.exe <команда>
```
Команды:\
`install` - установить службу\
`start` - запустить службу\
`stop` - остановить службу\
`remove` - удалить службу
